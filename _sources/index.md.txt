# hikari-flare

`flare` is a stateless component manager for hikari with type-safe API for use with [hikari](https://github.com/hikari-py/hikari)

Repository: [View on GitHub ](https://github.com/Lunarmagpie/hikari-flare)

PyPi: hikari-flare

If you need any help with this library or hikari, please do not hesitate to join the [hikari Discord](https://discord.gg/Jx4cNGG>)
